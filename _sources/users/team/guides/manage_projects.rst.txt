Users: Managing the Projects of a User
======================================

| A user can belong to many `projects </users/finance/guides/projects/project_general.html>`_
| For each project, a user is assigned a `role. Most users will simply be a project member,

#. Click :icon:`bars` in the ribbon
#. Click **Team**
#. Click **Users**
#. Click an active user
#. Click :icon:`ellipsis-v`
#. Click **Manage Projects**
#. Select a project from the displayed projects or find a project using the search feature

   * Click a project to assign the user to the project
   * By default, the user will be a member of the project
#. Change role by clicking the colored box to the right of the project's name

   * Click through three options (Member, Approver, and Owner) until the displayed option matches what role you would like to assign the user
#. Click :icon:`times` to the right of a row to remove the user from the project
#. Click **Save**
