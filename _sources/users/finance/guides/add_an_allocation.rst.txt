Adding an allocation
=================

.. note::
   When filling out certain forms to add check requests, reimbursements, etc., users will be offered the option to add an allocation. Users should do this when...

   Users should have arrived at a pop-up tab entitled **New Allocation**. Refer to the action's specific guide if you are having trouble arriving at this screen.

#. Choose a project
#. Choose an expense type
#. Describe the line item
#. Enter amount
#. Click **Done**
