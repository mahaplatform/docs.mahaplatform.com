Update Workflow ListsÂ¶

The Update Lists block allows you add or remove contacts from a list.
To use this block, you will need to have navigated to the workflow design tab. Click here if you are having trouble reaching this point.

  1. Place an Update Lists block in your workflow where desired

  2. Select Add to list or Remove from list

  3. Click Choose a list

  4. Select an existing list

  5. Alternatively, click Add New to create a new list (this list will automatically be chosen)

  6. Click Done
