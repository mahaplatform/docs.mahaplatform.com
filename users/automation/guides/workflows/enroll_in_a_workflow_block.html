Workflows: Adding an Enroll in Workflow BlockÂ¶

The Enroll in Workflow block allows you to enroll contacts in a separate workflow.
To use this block, you will need to have navigated to a workflow design tab. Click here to do so from Automation. Refer to Campaign and Phone guides for other workflows.

  1. Place an Enroll in Workflow block in your workflow where desired

  2. Click Choose a workflow

  3. Select an existing workflow

  4. Alternatively, click Add New to create a new workflow (this workflow will automatically be chosen)

  5. Click Done

  6. In the workflow, you can now click show more to display the workflow in which a contact will be enrolled
