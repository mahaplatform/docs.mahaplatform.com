Drive: Interacting with a File or a FolderÂ¶

To have access to these options, you must have navigated to the file or folder with which you would like to interact. Refer here if you are having trouble reaching this point.
The options available to you depend on whether you are interacting with a file or a folder, as well as what access you have.

  1. Click one of the following options to perform the associated action (click on hyperlinked options for guides)

    • Alternatively, click to access these options

  2. For a file you can view: Share File , Print File, Download File, View File

  3. For a file you can edit: All options above plus Manage Access , Manage Versions , Enable Sharing , Move File , Rename File , Remove File

  4. For a file you own: All options above

  5. For a folder you can view: No options available

  6. For a folder you can edit: Manage Access , Create New Folder , Upload File(s) , Move Folder , Rename Folder , Remove Folder

  7. For a folder you own: All options above
