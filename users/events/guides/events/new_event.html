New Event¶

Fill out the following information to create a new event - details for the event registration form,
To have access to the icon and to be able create a new event, you must be in the :icon:`calendar`**Upcoming Events** section of Events.

  1. Click the in the ribbon

  2. Click Events

  3. Click Events

  4. Click the in the upper right hand corner

  5. Choose a program

  6. Enter an event title

  7. Enter an event URL

  8. Describe the event

  9. Choose an image

  10. Click Add an organizer and select an organizer

    • You can also create a new organizer by clicking Add New. Enter a name, email address, phone number, and choose a photo, and click Save. This new organizer will automatically be added

    • If desired, add more organizers or click to delete a selected organizer

  11. Click Next

  12. Select a session or click Add a session and follow the steps below

    • Add a title

    • Enter a description

    • Choose a session location (if your session will be held virtually, check the box entitled This is an Online Session)

    • Select a date

    • Enter a start time and an end time (will format to standard time and default to AM unless otherwise specified)

    • Click Save

  13. Click Next

  14. Select a ticket type and select Allow pay at door if event-goers will not be required to pay for their ticket while registering

    • To create a new ticket type click Add a ticket type

    • If desired, add more ticket types

    • If desired, click to edit or click to delete a selected ticket type

  15. Click Next

  16. Enter contact’s first name

  17. Enter contact’s last name

  18. Enter contact’s email address

  19. If desired, click Add field and add an existing contact field or create and add a custom field

  20. Click Next

  21. Design ticket holder fields

    • By default, name on ticket, gender, age, race, and ethnicity will be added

    • You can disable these default fields (besides name on ticket) by clicking

    • You can add new ticket fields by clicking Add field, selecting an option, and clicking Done

  22. Create a confirmation email

    • Choose a sender

    • Enter the email addess customer’s will write to if they choose to reply to their confirmation email

    • Enter an email subject

    • Select an existing email template or create a new template

  23. Click Save
