Viewing and Editing an Event¶

To view all the different parts of an event and to make changes to them you will need to have navigated to the event. Click here if you are having trouble reaching this point.

  1. You will be brought to the :icon:`info-circle`**Details** page of the event you have selected

    • View event information, an audit trail, and comments

    • Click View Public Registration to open the event registration in a separate tab

    • Click and Edit Event to edit the event details or Clone Event to make a duplicate of an event

  2. Click :icon:`calendar-o`**Sessions**

    • View event sessions

    • Click and Edit Session to edit the session

    • Click in the upper right hand corner to create a new session

  3. Click :icon:`tags`**Ticket Types**

    • View ticket types

    • Click and Edit Ticket Type to edit the ticket type

    • Click and (De)activate Ticket Type to perform associated action

    • Click in the upper right hand corner to create a new ticket type

  4. Click :icon:`check-square`**Fields**

    • View contact fields and ticket fields by clicking Contact or Ticket

    • You can edit or delete a contact field by clicking (these options are not available for the contact’s first name, last name, or email)

    • You can add a contact field by clicking Add Field and add an existing contact field or create and add a custom field

    • You can disable any ticket fields (besides name on ticket) by clicking

    • You can add new ticket fields by clicking Add field and selecting an option

  5. Click :icon:`cogs`**Automation**

    • Click in the upper right hand corner to create a workflow

    • Click on a workflow to view details, emails, performance, edit the workflow, etc.

  6. Click :icon:`bullhorn`**Campaigns** WRITE AFTER YOU DO CAMPAIGNS *****

    • Click :icon:`envelope`**Email**

    • Click :icon:`comment`**SMS**

    • Click :icon:`phone`**Voice**

  7. Click :icon:`user-circle-o`**Registrations**

    • View contacts that registered for an event

    • Click a contact to view contacrt details, registration and revenue details, and ticket details moreeeee*****

  8. Click :icon:`tag`**Tickets**

    • View contacts that purchased a ticket

    • Click a contact to view a QR code, ticket holder details, event and attendance status

    • Click and click View eTicket to pull up a digital version of the contact’s unique ticket in a separate tab

  9. Click :icon:`check`**Attendance**

    • Click to ?????

    • Click and click Scan Tickets to scan tickets at an event

  10. Click :icon:`pie-chart`**Performance**

    • View ticket performance ??
