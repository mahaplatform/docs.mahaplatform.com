Commenting¶

You should navigate to the page on which you want to comment using the in the ribbon.
Comments can be viewed by anyone who has access to the feature you are commenting under.

  1. Click the bar entitled Write a comment… and type your comment

    • Add an emoji by clicking the and selecting an emoji

    • Add an image by clicking the and selecting an image

  2. If desired, upload a file

  3. Hit enter or return

  4. To reply to a comment, hover over the comment and click the

  5. To react to a comment, hover over the comment, click the and choose emoji

  6. To delete a comment, hover over the comment and click the
