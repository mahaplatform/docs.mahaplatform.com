Creating a New Invoice¶

Note

To bill a customer using the Maha Platform, you should create a new invoice.

  1. Click the <i class=”fa fa-bars”></i> menu icon in the ribbon

  2. Click Finance

  3. Click Revenue

  4. Click Invoices

  5. Click the <i class=”fa fa-plus”></i> plus icon in the upper right hand corner

  6. Choose a program

  7. Choose a customer

  8. Select the current date

  9. Select invoice due date

  10. Add notes

  11. Click Add Line Item

    • Choose a project

    • Choose a revenue type

    • Describe the item

    • Enter item unit price

    • Enter tax rate

    • Enter quantity

    • If item is tax deductible, check box

    • Click Done

  12. Add more line items if necessary

  13. Click Save
