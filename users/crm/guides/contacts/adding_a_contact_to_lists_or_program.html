Adding a Contact to a List or a ProgramÂ¶

When you create a contact, you will have the option to add them to lists or programs, as detailed in Adding a Contact
If you want to add an already created contact to a list or program, though, you should follow the steps below.

  1. Click the in the ribbon

  2. Click CRM

  3. Click Contacts

  4. Select contact(s) by clicking the to the left of the row * You can mass select all the contacts by clicking the in the upper left hand corner

  5. Click Add to Lists * Select lists to which you want to add the contact(s) * Click Save

  6. Click Add to Programs * Select programs to which you want to add the contact(s) * Click Save
