How to Reload
=============

.. image:: desktop/gridfunctions/reload.png
  :width: 400  ???????????????????????????

.. note::
   When the page needs to be refreshed (eg. page has timed out, a record has recently been added and has not yet appeared), you should reload the page.

   You should navigate to the page you wish to reload using the <i class="fa fa-bars"></i> **menu icon** in the ribbon.

#. Simply click the <i class="fa fa-refresh"></i> **refresh icon** in the right hand corner of the page
