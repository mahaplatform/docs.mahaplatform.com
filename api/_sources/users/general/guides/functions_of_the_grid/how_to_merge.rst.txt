Merging
=======

| When you want to merge already created vendors, contacts, etc., take the following steps. what can be merged?
| You should navigate to the page that contains what you wish to merge using the :icon:`bars` in the ribbon.

#. Click the :icon:`ellipsis-v` to the right of each row
#. Click **Merge**
#. Click blank field entitled **Merge Into**
#. Select or search for vendor, contact, etc.
#. Click **Merge**
