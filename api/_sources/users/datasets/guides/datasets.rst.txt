Datasets: Managing a Dataset
============================

|

#. Click :icon:`bars` in the ribbon
#. Click **Datasets**
#. Click a dataset to view the title, and audit trail, and comments
#. Click :icon:`ellipsis-v` and **Edit Dataset** to `edit </users/general/guides/functions_of_the_grid/how_to_edit.html>`_ the dataset title
#. Click :icon:`ellipsis-v` and **Create Type** to create dataset type

   * Enter a title
   * Click **Save**
 interact with the following features: `Access </users/datasets/guides/access.html>`_ , `API </users/datasets/guides/api.html>`_ , and any created dataset types
