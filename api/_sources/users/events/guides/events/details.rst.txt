Events: Managing Details
========================

| **Details** displays event information, an audit trail, and comments.
| To view this feature, you must have navigated to an event. Refer to `Events </users/events/guides/events/events.html>`_ if you are having trouble reaching this point.

#. You will be brought to the :icon:`info-circle` **Details** page of the event you have selected by default
#. Click **View Public Registration** to open the event registration in a separate tab
#. Click :icon:`ellipsis-v` and **Edit Event** to `edit </users/general/guides/functions_of_the_grid/how_to_edit.html>`_ the event details
#. Click :icon:`ellipsis-v` and **Clone Event** to make a duplicate of the event
