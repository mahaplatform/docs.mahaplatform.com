How to Edit
==========

.. note::
   When a user wants to make an edit to an already created account, contact, form, etc., then should follow the following steps.

   Users should navigate to the page that contains what they wish to edit using the <i class="fa fa-bars"></i> **menu icon** in the ribbon.

#. Click the <i class="fa fa-ellipsis-h"></i> **ellipsis icon** to the right of each row
#. Click **Edit**
#. Click on the field to edit it `(refer to specific guides if fields are unclear)`
#. Click **Save**
