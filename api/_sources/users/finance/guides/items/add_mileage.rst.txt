Add Mileage
===========

.. note::
   When a user makes a trip using business miles, they should record this by adding new mileage.
      * `If trips are frequent, consider using the **importing bulk mileage** feature`

#. Click the <i class="fa fa-bars"></i> **menu icon** in the ribbon
#. Click **Finance**
#. Click **Expenses**
#. Click **Items**
#. Click the <i class="fa fa-plus"></i> **plus icon** in the upper right hand corner
#. Click **Mileage**
#. Select date of trip
#. Describe the trip
#. Choose a project
#. Enter time leaving
#. Enter time arriving
#. Enter the odometer start
#. Enter the odometer end
#. Enter total miles
#. Click **Save**
