Domains: Adding a New Domain
============================

|

#. Click the :icon:`bars` in the ribbon
#. Click **Websites**
#. Click **Domains**
#. Click :icon:`plus` in the upper right hand corner
#. Click **Add a Domain**
#. Select an option:

   * Register Domain: Register a new domain name through Maha
   * Transfer Domain: Transfer an existing domain name to Maha
   * Point Domain: Point an externally registered domain name to Maha
#. Click **Next**
#. Proceed depending on the option you selected in step #6

   * Register Domain:
   * Transfer Domain:
   * Point Domain:
#. Click **Save**
