Workflows: Creating a Workflow via an Automation Tab
====================================================

| To manually create a workflow using this pathway, you will need to have navigated to an automation tab, for example, via `Programs </users/crm/guides/programs/programs.html>`_.

#. Select one of the trigger type(s) offered

   * For example, a workflow for `Lists </users/crm/guides/programs/lists.html>`_ offers you the option to select **Added to List** or **Removed from List**
   * When a contact is added to a list or removed from a list, depending on your selection, the contact will be enrolled in the workflow
#. Enter a title for the Workflow
#. You may choose for contacts to be enrolled in the workflow only once by clicking the checkbox
#. Click **Save**
