Dial Menu Block
===============

| The **Ask Question** block is only available when designing a workflow for a `voice campaign </users/campaigns/guides/voice/voice_campaigns.html>`_ or when managing a workflow for a `phone number </users/phone/guides/numbers/phone_numbers.html>`_ .

#. Place a **Dial Menu** block in your workflow where desired
#. Enter a menu name
#. Click `Speak text </users/automation/guides/workflows/speak_text_block.html>`_

   * Keep the greeting "Listen carefully to the following options" or write your own
#. Alternatively, click `Play an audio file </users/automation/guides/workflows/play_recording_block.html>`_
#. Click **Add Option**
#. Enter a number
#. Add a name
#. Click **Add**
#. To delete an option, click :icon:`times`
#. To edit </users/general/guides/functions_of_the_grid/how_to_edit.html>`_ an option, click :icon:`pencil`
#. If desired, click **Add Option** again
#. Click **Respond to hash (#)** or **Respond to star (*)** if you would like recipients to be able to press the hash and star
#. Click **Done**
